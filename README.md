# Test task Verba-Group
>Репозиторий содержит скрипт для парсинга сайта wildberries.ru по указанным в тестовых задачах фильтрам.
## Запуск
### Установка зависимостей
    pip install -r requirements.txt

Для работы скрипта требуется установленный на компьютере браузер [Chrome](https://www.google.com/intl/ru_ru/chrome/).
### Активация скрипта из папки проекта
    python src/app.py

## Результаты
Документ с тестовым заданием и ссылка на него хранятся в папке docs.
    
Результаты тестовых задач будут храниться в файле docs/WB products.xlsx. 
Результат первой задачи – в листе Quest 1, результат второй задачи – в листе Quest 2.

На данный момент в листах Quest 1 и Quest 2 файла WB products.xlsx хранится информация первых 300 продуктов, найденных по условиям каждой из тестовых задач.
